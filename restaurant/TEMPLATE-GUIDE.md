# Restaurant Template — How to Use

## To customise for a client, find and replace these placeholders:

| Placeholder | Replace with |
|---|---|
| `RESTAURANT NAME` | Client's restaurant name |
| `234XXXXXXXXXX` | Client's WhatsApp number (no + or spaces) |
| `STREET ADDRESS` | Client's street address |
| `AREA` | Client's area (e.g. Lekki, Ikeja, Surulere) |

## Images
- Hero background: replace Pexels URL with client's own food photo
- Menu preview cards: replace with client's actual dish photos
- About section: replace with client's kitchen/restaurant photo

## Menu
- Update dish names, descriptions and prices in `menu.html`
- Add or remove categories as needed
- Duplicate the dish row pattern for more items

## Google Maps
- In `contact.html`, replace the map placeholder div with the embed code from Google Maps
- Go to maps.google.com → search client address → Share → Embed a map → copy iframe code

## Colors
- Primary color is `#c2410c` (orange-red) throughout
- To change: find & replace `#c2410c` with client's brand color

## Tier
- This is a **Starter Package** template (3 pages)
- For Premium, add: Gallery page, About page, Reservation form
