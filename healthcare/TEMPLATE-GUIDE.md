# Healthcare / Clinic Template — How to Use

## To customise for a client, find and replace these placeholders:

| Placeholder | Replace with |
|---|---|
| `CLINIC NAME` | Client's clinic or hospital name |
| `234XXXXXXXXXX` | Client's WhatsApp number (no + or spaces) |
| `STREET ADDRESS` | Client's street address |
| `AREA` | Client's area (e.g. Surulere, Ikeja, Lekki) |
| `DOCTOR NAME` | Real names of each doctor (doctors.html) |

## Images
- Hero background (index.html): replace Pexels URL with a photo of the clinic exterior or interior
- Doctors page: replace Pexels doctor photos with real staff headshots

## Services
- Edit the services in `services.html` to match what the clinic actually offers
- Remove any services the clinic doesn't provide (e.g. dental, eye care)
- Add new services by duplicating the existing card pattern

## Doctors
- Update all 6 doctor cards in `doctors.html` with real names, specialties, and qualifications
- Replace Pexels photos with real staff photos
- Remove or add doctor cards depending on team size — duplicate the card pattern to add more

## Clinic Hours
- Update opening hours in `appointments.html` and `contact.html` to match actual hours

## HMO Plans
- The services page mentions HMO — update or remove this if the client doesn't accept HMO

## Google Maps
- In `contact.html`, replace the map placeholder div with the embed code from Google Maps
- Go to maps.google.com → search client address → Share → Embed a map → copy iframe code

## Colors
- Primary navy: `#0f3460`
- Accent teal: `#0d9488`
- To change: find & replace both values with client's brand colors

## Pages in This Template
| Page | Description |
|---|---|
| `index.html` | Home — hero, stats, 6 service cards, why us, CTA |
| `services.html` | Detailed services (8 services with full bullet points) |
| `doctors.html` | 6 doctor profiles with Book Appointment buttons |
| `appointments.html` | 3-step booking guide, big WhatsApp button, clinic hours |
| `contact.html` | WhatsApp CTA, location, hours, emergency note, map |

## Tier
- This is a **Premium Package** template (5 pages)
- For VIP, add: Patient Reviews page, Health Blog, Insurance/HMO page
