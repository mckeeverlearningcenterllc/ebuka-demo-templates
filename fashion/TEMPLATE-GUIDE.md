# Fashion / Boutique Template — How to Use

## To customise for a client, find and replace these placeholders:

| Placeholder | Replace with |
|---|---|
| `BOUTIQUE NAME` | Client's boutique or brand name |
| `234XXXXXXXXXX` | Client's WhatsApp number (no + or spaces) |
| `STREET ADDRESS` | Client's street address |
| `AREA` | Client's area (e.g. Lekki, VI, Surulere) |

## Images
- Hero background (index.html): replace Pexels URL with client's own brand or lookbook photo
- Collections page: replace Pexels URLs with client's actual outfit photos
- About page: replace studio photo with client's workshop or brand shoot

## Collections
- Update collection names, descriptions, and photos in `collections.html`
- Add or remove collection sections (Ankara, Bridal, Ready to Wear) as needed
- Duplicate any item card for more products

## About Page
- Update the story text in `about.html` to reflect the client's real background
- Update stats (clients served, years in business, etc.) to match client's real numbers

## Google Maps
- In `contact.html`, replace the map placeholder div with embed code from Google Maps
- Go to maps.google.com → search client address → Share → Embed a map → copy iframe code

## Colors
- Primary color is `#1a0a2e` (deep purple) and accent is `#d4a843` (gold) throughout
- To change: find & replace `#1a0a2e` and `#d4a843` with client's brand colors
- Secondary dark card color `#2a1a3e` should also be updated if changing the purple

## Pages in This Template
| Page | Description |
|---|---|
| `index.html` | Home — hero, featured collections, why choose us, CTA |
| `collections.html` | Full collections — Ankara, Bridal & Events, Ready to Wear |
| `about.html` | Brand story, values, stats |
| `process.html` | 5-step ordering process + measurements guide |
| `contact.html` | WhatsApp CTA, address, hours, Google Maps |

## Tier
- This is a **Premium Package** template (5 pages)
- For VIP, add: Gallery/Lookbook page, Blog/Style Tips, Size Guide page
