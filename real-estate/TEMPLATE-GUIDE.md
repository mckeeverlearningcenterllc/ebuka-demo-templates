# Real Estate / Property Agency Template — How to Use

## To customise for a client, find and replace these placeholders:

| Placeholder | Replace with |
|---|---|
| `AGENCY NAME` | Client's agency or business name |
| `234XXXXXXXXXX` | Client's WhatsApp number (no + or spaces) |
| `STREET ADDRESS` | Client's office street address |
| `AREA` | Client's area (e.g. Lekki, VI, Ikeja) |
| `AGENT NAME` | Real names of team members (about.html) |

## Images
- Hero background (index.html): replace Pexels URL with client's own property or office photo
- Listings: replace Pexels URLs with actual photos of each property
- About/Team: replace with real photos of the client's team

## Listings
- Update property names, descriptions, prices, and locations in `listings.html`
- Change badge color for "For Rent" vs "For Sale" as needed:
  - For Sale: `style="background:#c8922a; color:#162032;"`
  - For Rent: `class="bg-green-500 text-white"`
- Add or remove property cards by duplicating the card pattern
- Update the WhatsApp message text in each "Enquire" button to match the specific property

## Team Section (about.html)
- Replace all 3 `AGENT NAME` placeholders with real names and titles
- Replace Pexels team photos with real staff photos

## Google Maps
- In `contact.html`, replace the map placeholder div with the embed code from Google Maps
- Go to maps.google.com → search client address → Share → Embed a map → copy iframe code

## Colors
- Primary dark: `#162032` (deep slate)
- Accent gold: `#c8922a`
- Secondary dark card: `#1e2e44`
- To change: find & replace all three values with client's brand colors

## Pages in This Template
| Page | Description |
|---|---|
| `index.html` | Home — hero, stats bar, 3 featured listings, why us, CTA |
| `listings.html` | Full listings — For Sale (4 properties) + For Rent (2 properties) |
| `about.html` | Story, values, stats, 3-person team section |
| `services.html` | 4 services: Sales, Rentals, Management, Valuations + How It Works |
| `contact.html` | WhatsApp CTA, office info, hours, Google Maps |

## Tier
